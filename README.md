# LifePlan AI Australia

A mobile-first Australian financial planning tool.

## Core experience

- Enter monthly after-tax income
- Generate a Barefoot-inspired budget automatically
- Switch goals: balanced, home deposit, FIRE, travel, or family
- Adjust the savings rate with a what-if slider
- Project future super using a Hostplus-style benchmark
- Compare conservative, balanced, and growth return assumptions
- Reset and start again at any time

## Super assumptions

The projection currently uses:

- 12% employer Super Guarantee
- 15% contributions tax
- 2.5% inflation
- User-selected investment return

This is an educational planning tool and not personal financial advice.
